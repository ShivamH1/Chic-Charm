## E-Commerce Website

A Simple E-commerce site where you get everything. From Men's clothing to electronics you will find here.

## Installation

Clone the repository
```bash
[git clone https://github.com/ShivamH1/Transaction-App.git](https://github.com/ShivamH1/The-Little-Things.git)
```

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install 
```

Then 
```bash
npm start / npm run dev
```
